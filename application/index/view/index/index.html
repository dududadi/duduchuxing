滴滴完成超55亿美元新一轮融资 布局国际化与前沿智能交通科技

4月28日，全球领先的出行平台滴滴出行今日宣布完成新一轮超过55亿美元融资，以支持其全球化战略的推进和前沿技术领域的投资。

程维联合国英文演讲：滴滴已成立全球新能源汽车服务公司

美国东部时间11月1日，滴滴创始人、董事长兼CEO程维在“建设全球能源互联网，助推联合国2030可持续发展”论坛上表示，滴滴已经携手全球能源互联网发展合作组织（GEIDCO）合资成立全球新能源汽车服务公司。
